# Cypress Testing

Cypress + Cucumber framework (Reference: https://www.toolsqa.com/cypress/bdd-automation-framework-in-cypress/)

## Installation

Install all the dependancies using npm

```bash
npm install
```

## Usage

```bash
npx cypress open
```

Then click on the feature file you want to run
